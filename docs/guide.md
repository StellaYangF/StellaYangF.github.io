# 前言

博客记录的初衷，想让时间留下痕迹，技术发展层出不穷，万变不离其宗，技术需要深入底层原理。身为技术人，享受钻研技术、破解奥秘的快乐是巨大的。

生活中，也需要热情，热爱生活，爱上阅读，以滋养充实大脑灵魂。
